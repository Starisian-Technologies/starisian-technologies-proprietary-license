# Security Policy

## Reporting a Vulnerability

Please do **not** report security vulnerabilities through public GitHub issues.

If you discover a potential security risk or a data leak within our linguistic datasets or marketing systems, please report it privately using GitHub Private Vulnerability Reporting / the repository Security Advisories feature. If that is unavailable, contact the security team at **security@example.com**.

Please include enough detail to reproduce the issue, and do not disclose vulnerability details publicly until we have investigated and provided guidance.

We aim to respond to all security concerns within **48 hours**.
